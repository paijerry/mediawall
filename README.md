# mediawall

[連結](https://paijerry.github.io/mediawall/){:target="_blank"}

拖放圖片與影片，並可以縮放移動，影片可改音量與播放速度  
可以使用參數決定分割方式


## 分割方式

v 縱向切  
h 橫向切

例:  
?v=2-1-1 左佔1/2, 中佔1/2, 右佔1/2  
?h=1-1 上下各半  
?v=2-1-1&h=1-1 照上方式切成六份  

## 輪播

支援在同一格拖入多個圖片與影片

混合輪播：一次拖入多個檔案(圖片＋影片混合也可以)，就會依順序循環播放。  
切換規則：圖片顯示「間隔」秒數後換下一個；影片播完一輪才換下一個（不再單獨循環）。  
控制列三合一：滑鼠移到格子上會看到「間隔」（1～10 秒，管圖片停留時間）、「音量」（預設 0%）、「速度」三支拉桿。音量和速度套用到輪播中的每支影片，中途調整也會立即作用到正在播的影片。  
縮放記錄照常：每張圖、每支影片各自的縮放與位移狀態都會記住，輪回來時自動還原。
